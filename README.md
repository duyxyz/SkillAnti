# DuyXYZ Skills Store

Welcome to the specialized skills repository for developers and AI Agents. This repository contains clean code standards, DevOps operational processes, and professional automation tools.

## Quick Install

To add this skill set to your project, run the following command:

```bash
npx skills add duyxyz/skills
```
