# 🚀 DuyXYZ Skills Store

Chào mừng bạn đến với kho lưu trữ kỹ năng (skills) chuyên biệt dành cho lập trình viên và AI Agent. Kho lưu trữ này chứa các tiêu chuẩn mã nguồn sạch, quy trình vận hành DevOps và các công cụ tự động hóa chuyên nghiệp.

## 🛠️ Cài đặt nhanh

Để thêm bộ kỹ năng này vào dự án của bạn, hãy chạy lệnh sau:

```bash
npx skills add duyxyz/skills
```
